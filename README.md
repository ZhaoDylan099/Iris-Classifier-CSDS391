# Iris Classifier
CSDS 391 Introduction to AI assignments

## Manual MLP
Contains code and resulting iterations of decision boundaries for manual implementation of an MLP model for classifying the iris dataset. Read ZhaoDylan_HW9.pdf for more information.

## Scikit-Learn MLP Implementation and K-means
Contains code implementing MLP and K-means using Scikit-Learn and comparing the performance between the two models as well as evaulating perfomance differences between supervised and unsupervised models. Read ZhaoDylan_HW9_bonus.pdf for more information.
